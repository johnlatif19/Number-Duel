# 🎯 Number Duel - لعبة تخمين الأرقام متعددة اللاعبين

![Number Duel Logo](https://i.postimg.cc/tJB1SG8c/Number-Duel.png)

## 📝 عن اللعبة

**Number Duel** هي لعبة تخمين أرقام متعددة اللاعبين حيث يتنافس لاعبين ضد بعضهما البعض. كل لاعب يختار رقم سري ويحاول تخمين رقم الخصم. اللعبة تدعم الدردشة الفورية وتجربة تفاعلية ممتعة.

### 🎮 آلية اللعب

1. كل لاعب يختار رقم سري (يمكن أن يكون أي عدد صحيح: موجب، سالب، أو صفر)
2. اللاعبون يتناوبون في التخمين
3. بعد كل تخمين، يحصل اللاعب على تلميح:
   - 🔼 أعلى (إذا كان الرقم أكبر من رقم الخصم)
   - 🔽 أقل (إذا كان الرقم أصغر من رقم الخصم)
   - ✅ صحيح (إذا كان التخمين صحيح)
4. الفائز هو من يخمن رقم الخصم بشكل صحيح
